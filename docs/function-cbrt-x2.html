Description & Behavior

This function has a cusp at x = 0, which is a local minimum. Unlike the cube root's vertical tangent, no tangent line is defined at the cusp.

What the Derivatives tell us

The cusp at x = 0 is a critical point and local minimum where one-sided derivatives are infinite with opposite signs — no tangent line exists. The origin is also an inflection point where concavity changes.
