Description & Behavior

This seemingly very simple function is an important example of a function where every point is a critical point, and every point is the location of an absolute max, absolute min, local max, and local min. Because its slope is zero, it is neither increasing nor decreasing anywhere. Because it is a straight line, it has no concavity.
