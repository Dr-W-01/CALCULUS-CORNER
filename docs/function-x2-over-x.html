Description & Behavior

This function looks like f(x) = x but has a removable discontinuity at x = 0. Because of the hole, there is no minimum value even though values approach 0.

Limit at a Point

General Limit

This function is continuous on its domain, but has a removable discontinuity at x = 0.

What the Derivatives Tell Us

On its domain, f'(x) = 1 > 0 so the function increases everywhere with no critical points. The hole at x = 0 prevents a witness point for any minimum near the origin.
