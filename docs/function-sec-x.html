Limits and Continuity

End Behaviours

Secant oscillates between unbounded growth and its minimum value of 1, with no limit at ±∞.

General Limit

This function is continuous on each interval of its domain, with infinite discontinuities at x = π/2 + kπ.

What the Derivatives Tell Us

Local minima of 1 occur at x = kπ where cos is ±1. The first derivative changes sign at these points while the second derivative stays positive on each branch — always concave up.
