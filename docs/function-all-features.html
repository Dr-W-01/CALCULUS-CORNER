Limits and Continuity

End Behaviours

Near x = -4 the reciprocal factor dominates and the function blows up. At the left endpoint x = -10 the square root factor forces g(x) → 0.

Limit at a Point

The hole at x = 0 has a finite two-sided limit but no function value. The jump at x = -1 and the asymptote at x = -4 are the other headline limit behaviors.

This function is continuous on each interval of its domain but has a removable discontinuity at x = 0, a jump at x = -1, an infinite discontinuity at x = -4, and an endpoint at x = -10.
