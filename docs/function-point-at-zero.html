Limits and Continuity

End Behaviours

The domain is a single point, so end behaviour is not applicable.

Limit at a Point

The function is defined only at x = 0. Limits from either side do not apply in the usual sense because the domain has no interval around 0.

The function is continuous at x = 0 on its domain {0}, with endpoint behavior relative to any larger interval.

What the Derivatives Tell Us

With domain {0}, the sole point is a critical point but there are no intervals of increase or decrease. Derivatives are undefined because there is no open interval containing 0 in the domain.
