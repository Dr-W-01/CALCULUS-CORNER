Description & Behavior

Hyperbolic secant is 1/cosh(x) — an even, bell-shaped function with maximum value 1 at x = 0, approaching 0 as x → ±∞.

What the Derivatives Tell Us

The derivative changes from positive to negative at x = 0, giving a relative (and absolute) maximum of 1. The second derivative changes sign at x = ±arcsinh(1), producing inflection points on each side of the peak.
